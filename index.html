爬蟲成果展示

本專案使用 Python 撰寫爬蟲，從 PTT 八卦板擷取文章資訊，包含標題、作者與推文數。以下展示整體流程、程式碼重點、資料範例與圖表成果。

📋 爬蟲流程

  1. 用 requests 送出 GET 請求（含成年確認）
  2. 用 BeautifulSoup 解析 HTML 結構
  3. 擷取每篇文章的標題、作者、推文數
  4. 資料儲存成 CSV，進行後續視覺化

💻 核心程式碼片段


import requests
from bs4 import BeautifulSoup

url = "https://www.ptt.cc/bbs/Gossiping/index.html"
res = requests.get(url, cookies={'over18': '1'})
soup = BeautifulSoup(res.text, 'html.parser')

for post in soup.select('.r-ent'):
    title = post.select_one('.title').text.strip()
    author = post.select_one('.author').text.strip()
    push = post.select_one('.nrec').text.strip()
    print(title, author, push)
